1 两数之和

public  static int[] twoSum(int[]  nums, int target) {
    int[] results = {0,0}; //记录结果
    int i=0,j=0;
    outer:for (i=0;i<nums.length - 1;i++){
        inner:for (j = i+1;j<nums.length;j++){
            if (nums[i] + nums[j] == target){
                break outer; //跳出全部的循环
            }
        }
    }
    results[0] = i;
    results[1] = j;
    return results;
}

public static void main(String[] args) {
    int[] n = {2, 7, 11, 15};
    int target = 9;

    int[] result  = twoSum(n,target);
    for (int r:result) {
        System.out.println(r);
    }
}

2 无重复字符的最长子串

java中的集合set， 里面的字符都是唯一的，即没有重复的，所以可以检测里面一个字符串总有没有重复的字符。在进行添加到集合之前，进行检测里面有没有该字符。

Set<Character> set = new HashSet<>();
set.contains(ch) //判断集合中是否存在字符ch。存在则返回true。
  1. 暴力求解的方法
//使用双重循环进行操作，即全遍历然后使用set进行检测是否有重复的字符。 

public static int lengthOfLongestSubstring(String s) {
        int i=0,j=0;
        int max = 0;
        for (int k = 0; k < s.length()-1; k++) {
            for (int l = k + 1 ; l < s.length() ; l++) {
                if (isUnique(s,k,l)){
                    max = Math.max(max,l-k); //找出最大值
                }
            }
        }
        return max;
    }

    // 使用集合找出字符串s中的相同的东
    public static boolean isUnique(String s, int start,int end){
        Set<Character> set = new HashSet<>();
        for (int i = start; i < end ;i ++){
            char ch =s.charAt(i);
            if (set.contains(ch)) return  false;
            set.add(ch);
        }
        return true;
    }
  1. 使用滑动窗口进行解题
public int lengthOfLongestSubstring(String s) {
    int n = s.length();
    Set<Character> set = new HashSet<>();
    int ans = 0, i = 0, j = 0;
    while (i < n && j < n) { //检测到最后即可跳出循环。
        if (!set.contains(s.charAt(j))){ 
            set.add(s.charAt(j++));   //当添加到一个值时，则窗口直接加一。j++
            ans = Math.max(ans, j - i); 
        }
        else {
            set.remove(s.charAt(i++));// 删除以后，窗口缩小一位。即i++
        }
    }
    return ans;
}

即使用滑动窗口进行检测，窗口中总是保证元素是不同，在检测到有相同元素的时候，将前面的进行删除，增加后者，直到检测到最后一个值便进行结束循环，只需要简单的测试就行。
